# This is fullstack project with clien part and server part

The project is implemented on React js.
To see the frontend or backend, go to the appropriate branch.

## Frontend

The following technologies have been used:

- SCSS
- module.scss
- Axios
- React router dom
- Redux toolkit
- Implemented authorization on the site

### Backend

The following technologies have been used:

- Nodemon
- Cors
- Express
- Mongoose
- Bcrypt js
